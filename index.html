Install ZenSkin

One command to transform your Linux desktop

terminal
$ # Download and install ZenSkin (Beta)
wget -O zenskin.deb https://github.com/bhuvanesh-m-dev/zenskin/releases/download/zenskin-v1.deb/zenskin-v1.deb && sudo dpkg -i zenskin.deb || (sudo apt --fix-broken install -y && sudo dpkg -i zenskin.deb) && zenskin_
Auto-launches after install ~2-5 minutes
